### Trigger

The `Trigger` class enables you to access details about a running trigger.

#### Properties

- **isDebugEnabled: boolean**: The value is

`true`

if debugging is enabled for the trigger. The value is determined by the state of the

Enable debug log

checkbox in the Edit Trigger pane in the ExtraHop system.
